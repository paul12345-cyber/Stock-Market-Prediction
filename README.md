## Indian Equity Market Prediction with Linear Regression


Dataset (2000 -2010) for our model training was obtained from [Quandl](https://www.quandl.com/data/TC1/HDFC)

Goto [Quandl](https://www.quandl.com/) and create an account to obtain API key to be used
in downloading dataset

Install Quandl from anaconda prompt: ```pip install Quandl```
